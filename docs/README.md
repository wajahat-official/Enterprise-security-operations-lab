# Portfolio Documentation

This directory contains the complete Enterprise Security Operations Portfolio developed as part of a hands-on cybersecurity lab environment.

## Portfolio Overview

The portfolio demonstrates the deployment, configuration, monitoring, and security operations of a Windows-based enterprise environment integrated with Splunk SIEM.

### Technologies Used

* Active Directory Domain Services (AD DS)
* Windows Server
* DNS & DHCP
* Group Policy Objects (GPO)
* File Server & NTFS Permissions
* Windows Event Forwarding (WEF)
* Microsoft Defender for Endpoint
* Splunk Enterprise SIEM
* Security Monitoring & Detection Engineering
