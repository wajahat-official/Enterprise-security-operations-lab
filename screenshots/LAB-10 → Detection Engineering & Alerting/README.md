# LAB-10 – Detection Engineering & Alerting

## Overview

This lab focused on implementing automated detection and alerting workflows using Splunk Enterprise. Multiple security alerts were developed and validated to support proactive monitoring and incident response activities.

## Objectives

* Develop detection rules
* Create automated security alerts
* Configure alert scheduling
* Generate security reports
* Monitor security events through dashboards

## Technologies Used

* Splunk Enterprise
* SPL (Search Processing Language)
* Windows Security Logs
* Active Directory

## Tasks Performed

### 1. Failed Logon Detection

Created automated alerts for repeated failed authentication attempts.

### 2. Account Lockout Detection

Configured alerting for account lockout events.

### 3. User Creation Detection

Implemented monitoring and alerting for newly created user accounts.

### 4. Group Membership Monitoring

Configured alerts for security group membership changes.

### 5. Reporting & Visualization

Created security reports and dashboard visualizations.

### 6. Alert Validation

Validated detection rules using generated security events.
