# LAB-09 – Threat Detection & Security Monitoring Dashboard

## Overview

This lab focused on developing practical security monitoring use cases using Windows Security Event Logs and Splunk Enterprise. Security dashboards were created to visualize authentication activity and account management events.

## Objectives

* Develop security monitoring use cases
* Investigate Windows Security Event IDs
* Create SPL searches
* Build security monitoring dashboards
* Validate detection logic

## Technologies Used

* Splunk Enterprise
* SPL (Search Processing Language)
* Windows Security Logs
* Active Directory

## Tasks Performed

### 1. Log Verification

Verified successful ingestion of Windows Security Logs.

### 2. Authentication Monitoring

Monitored successful and failed authentication events.

### 3. Account Management Monitoring

Monitored user account creation and group membership changes.

### 4. Dashboard Development

Created centralized security monitoring dashboards.

### 5. Detection Validation

Validated monitoring logic using real security events.

## Event IDs Investigated

* 4624 – Successful Logon
* 4625 – Failed Logon
* 4720 – User Account Created
* 4728 – User Added to Security Group
* 4732 – User Added to Local Group
* 4740 – Account Lockout

## Key Skills Demonstrated

* Threat Detection Fundamentals
* Security Monitoring
* SPL Development
* Dashboard Creation
* Security Event Analysis

## Outcome

Successfully developed multiple security monitoring use cases and implemented a centralized security monitoring dashboard using Splunk Enterprise.
