# LAB-06 – Microsoft Defender Security Monitoring

## Overview

This lab focused on validating endpoint security monitoring and threat detection capabilities using Microsoft Defender. Security events were generated and analyzed to verify that endpoint protection mechanisms were functioning correctly within the enterprise-style lab environment.

## Objectives

* Validate Microsoft Defender functionality
* Generate security-related events
* Perform malware detection testing
* Verify endpoint protection capabilities
* Monitor Defender security events

## Technologies Used

* Microsoft Defender Antivirus
* Windows Security Center
* Windows Event Viewer
* Windows Security Logs
* EICAR Test File

## Tasks Performed

### 1. Defender Configuration Review

Verified that Microsoft Defender was enabled and actively protecting the Windows endpoint.

### 2. Endpoint Security Monitoring

Reviewed Microsoft Defender security settings and monitored protection status.

### 3. Malware Simulation Testing

Performed a controlled malware simulation using the EICAR test file to validate Defender detection capabilities.

### 4. Threat Detection Validation

Confirmed that Microsoft Defender successfully detected and responded to the simulated threat.

### 5. Security Event Verification

Reviewed security logs and Defender-generated events to validate proper logging and alert generation.
