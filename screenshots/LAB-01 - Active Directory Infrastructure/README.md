# LAB-01 – Active Directory Infrastructure

## Overview

This lab focused on deploying and configuring the foundational Active Directory infrastructure required for an enterprise Windows environment.

## Objectives

* Install Active Directory Domain Services (AD DS)
* Promote Windows Server to a Domain Controller
* Create Organizational Units (OUs)
* Create and manage users and security groups
* Verify domain functionality

## Technologies Used

* Windows Server
* Active Directory Domain Services (AD DS)
* DNS
* Organizational Units (OUs)
* User & Group Management

## Tasks Performed

### 1. Active Directory Installation

Installed the Active Directory Domain Services role on the Windows Server.

### 2. Domain Controller Configuration

Promoted the server to a Domain Controller and created a new forest and domain.

### 3. Organizational Unit Structure

Created Organizational Units to organize users, groups, and computer objects.

### 4. User and Group Management

Created domain users and security groups following enterprise administration practices.

### 5. Validation

Verified domain functionality, authentication services, and Active Directory management tools.

## Key Skills Demonstrated

* Active Directory Administration
* Domain Controller Deployment
* User Lifecycle Management
* Group Administration
* Enterprise Identity Management

## Outcome

Successfully deployed a functional Active Directory environment that serves as the foundation for all subsequent security monitoring and security operations labs.
